# App Access Instructions — For Google Play Review Team

## Access type
Restricted — requires pairing with another device running Aegis.

## Test instructions
1. Install Aegis on two test devices
2. Open Aegis on Device A → tap "Add Contact" → generates QR code
3. Open Aegis on Device B → tap "Add Contact" → "Scan QR" → scan Device A's code
4. Devices are now paired. Messaging works between them.
5. Panic test: On Device A, hold the shield icon for 3 seconds
6. Device B receives panic alert with location

## No server required
Aegis is peer-to-peer. No login, no account, no server to configure.
Both devices must have internet access (WiFi or cellular).

## Demo credentials
Not applicable — no accounts exist. Pairing is cryptographic (QR code exchange).
