# Content Rating Questionnaire — Aegis

## IARC Rating: Expected result — Everyone / PEGI 3

### Violence
- Does the app contain violence? **No**
- Does it depict violence toward a specific race, ethnicity, etc.? **No**
- Does it contain realistic violence? **No**

### Sexuality
- Does the app contain sexual content? **No**
- Does it contain nudity? **No**

### Language
- Does the app contain profanity? **No**
- Does it contain crude humor? **No**

### Controlled substances
- Does the app reference drugs, alcohol, or tobacco? **No**

### Interactive elements
- Does the app allow users to interact/communicate? **Yes**
  - Type: Unrestricted peer-to-peer communication (encrypted messaging)
- Does it share user location? **Yes** (with paired contacts only, by explicit pairing)
- Does it allow purchases? **No**

### Miscellaneous
- Does it contain gambling? **No**
- Is it a news app? **No**
- Does it contain user-generated content? **Yes** (messages, photos between users)

## Notes for reviewer
Aegis is a personal security app. All communication is between explicitly paired contacts.
There is no public content, no discovery, no stranger interaction.
The panic system involves location sharing and audio/camera — these are emergency features,
not surveillance. Both parties must explicitly consent to pairing.
