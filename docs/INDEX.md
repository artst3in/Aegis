# Aegis documentation

Aegis is a personal-security messaging app: end-to-end encrypted chat built
on the SimpleX network, panic broadcast to chosen contacts, and unattended-
phone monitoring — built for people for whom phone security is not optional.

## Detailed design specifications

The full protocol and feature design specifications are maintained
privately. Some describe protective mechanisms whose internals are
deliberately not published, so that disclosure cannot help an adversary
defeat them. For questions about the security design, use the contact in
[SECURITY](../SECURITY.md).
